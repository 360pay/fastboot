putty小技巧

2017-02-02

安装putty完设置好windows系统环境变量之后
可以打开CMD窗口
直接执行以下命令.

通过telnet协议连接
putty -telnet 1.1.1.1

通过ssh协议连接
putty -ssh 1.1.1.109 -P 11021 -l root -pw 416bcbe8af2704b27d508977bc

通过载入一个putty已保存的session来连接
putty -load "cacti" -pw 416bcbe8af2704b27d508977bc
